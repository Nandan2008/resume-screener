> An end-to-end AI system that analyzes resumes against job descriptions using real Machine Learning and NLP.

## 🧠 How It Works
## ⚡ Tech Stack
| Layer | Technology |
|---|---|
| Frontend | React + Vite + Tailwind CSS |
| Backend | Python + FastAPI |
| PDF Parsing | PyMuPDF |
| NLP | spaCy + NLTK |
| ML Matching | TF-IDF + Cosine Similarity |
| Semantic AI | Sentence Transformers |
| AI Advisor | Google Gemini 1.5 Flash |

## 🚀 Features
- 📄 PDF resume upload with drag & drop
- 🧠 NLP entity extraction
- 📊 Dual scoring system
- 💡 AI career advice
- 📚 Skill gap analysis
